## Features

<Card title="General Features">
    - มีระบบร้านค้าสัตว์เลี้ยงในระบบ (สามารถปิดได้)
    - มีระบบร้านค้าอาหารในระบบ (สามารถปิดได้)
    - สามารถเลี้ยงเเบบไอเท็มได้
    - ในเเต่ละจุด Positions สามารถเลี้ยงสัตว์มากกว่า 1 ชนิดขึ้นไปได้
    - มีระบบเควส Missions Get ที่ผู้เล่นต้องทําเควสเลี้ยงสัตว์ตามจํานวนที่ตั้ง ในตัวได้
    - ระบบการให้อาหารสามารถตั้งค่าเเรนดอมได้มากกว่า 1 กรณีที่สัตว์เลี้ยงหิวสามารถตั้งค่าอาหารได้มากกว่า 1 ชนิด
</Card>

<Card title="Automatic Features">
    - การให้อาหารออโต้ (เลี้ยงออโต้)
    - มีไอเท็มสําหรับ VIP ในการเพิ่มลิมิตจํานวนในการเลี้ยงได้
</Card>

<Card title="User Interface (UI)">
    - UI มีการบอกสถานะของสัตว์เลี้ยงอย่างชัดเจน
    - UI Mini ไม่รกตาเมื่อพับจอหรือปิดเมนูหลัก
    - UI ที่ไหลลื่น สวยทันสมัย มีอนิเมชั่น
    - UI สามารถปรับเเต่งสีเเละคัสตอมเพิ่มได้!
</Card>

<Card title="Security Features">
    - ป้องกันการเสกไอเทม (TriggerServerEvent)
    - ป้องกันการเลี้ยงสัตว์เร็วกว่ากําหนด (ใช้โปรเเกรมเลี้ยงไวหรือปรับเวลา)
    - ไม่สามารถเสกไอเทมหรือเเอดไอเทมมากกว่า Limit จากคอนฟิกที่ตั้งได้ เช่น ตั้งให้เลี้ยง 5 ตัว Request Get ได้เเค่ 5 ครั้งเท่านั้น
</Card>

## Where to Setting?

<Columns cols={2}>
    <Card title="config.default.lua" icon="code">
        สำหรับการตั้งค่าเบื้องต้นเช่น การเปิด Debugger, ตั้งค่าปุ่มเปิด UI, ตั้งค่า Event เมื่อผู้เล่นเสียชีวิตและตั้งค่าโซนสำหรับการเลี้ยงสัตว์
    </Card>

    <Card title="config.feeds.lua" icon="code">
        สำหรับการตั้งค่าการให้อาหารแต่ละโซน (โซนตั้งได้ที่ config.default.lua) เช่นไอเทมที่ใช้ได้ ราคาอาหาร และคำอธิบาย
    </Card>

    <Card title="config.framework.lua" icon="code">
        สำหรับการตั้งค่าเพื่อให้ใช้งานกับ Framework ได้ (ESX)
    </Card>

    <Card title="config.lists.lua" icon="code">
        สำหรับการตั้งค่า Ped หรือสัตว์เลี้ยงแต่ละโซน เช่นโซนนี้ให้มีสัตว์ชนิดไหน และใช้ไอเทมอะไรในการให้อาหาร
    </Card>

    <Card title="config.notify.lua" icon="code">
        สำหรับการตั้งค่าข้อความแจ้งเตือนต่าง ๆ และปรับแต่งการเลือกใช้ฟังก์ชั่นในการแจ้งเตือน
    </Card>
</Columns>
