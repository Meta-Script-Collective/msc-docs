## Features

<Card title="General Features">
    - สามารถเปลี่ยนอาวุธบนรถได้
    - เมื่อเปลี่ยนอาวุธเเล้ว ผู้เล่นทุกคนจะเอฟเฟคสกินตรงกันเเละเอฟเฟคหน้าจอสกิน (ถ้ามี)
    - เมื่อผู้เล่นมีการใช้งานสกินอาวุธกรณีที่อาวุธหลักหายจะทําการลบอาวุธ Skin ให้อัตโนมัติ
</Card>

<Card title="User Interface (UI)">
    - ระบบ UI มีการบอกสถานะของอาวุธอย่างชัดเจน (พร้อมใช้งาน , ใช้งาน , ไม่พร้อมใช้งาน)
    - ระบบ UI มีการบอกค่าดาเมจ (เปิด ปิดได้ ใช้ Export ได้จากสคริปต์ดาเมจ) มี (ดาเมจเข้าเลือด, ดาเมจเข้าเกราะ, อัตราการคริ, ดาเมจติดพิษ)
    - ระบบ UI ที่ไหลลื่น สวยทันสมัย มีอนิเมชั่น
    - ระบบ UI สามารถปรับเเต่งสีเเละคัสตอมเพิ่มได้!
    - เมื่อเปลี่ยนอาวุธเเล้ว สถานะบน UI จะอัพเดตเเบบ Real-Time !
    - สามารถทดลองเอฟเฟคผ่าน UI ได้
</Card>

<Card title="Weapon Skin System">
    - ระบบ รองรับอาวุธที่เป็น Components
    - ระบบ การเก็บข้อมูลสกินอาวุธเเยกเเละใช้งานง่ายมีการเเยกเก็บข้อมูลที่ชัดเจน
    - ระบบ สคริปต์ปลอดภัย เเละลื่นไหล
    - Config สามารถเพิ่มอาวุธเเละสกินได้ไม่จํากัด
    - Config สามารถตั้งค่าโหมดทดลองอาวุธได้
</Card>

<Card title="Customization Bridge">
    - Export ใช้งานได้กว้าง
    - มีคู่มือการใช้งานที่ชัดเจน
    - สามารถเขียนสคริปต์เพิ่มเติมได้ ทั้งฝั่ง Client และ Server
    - สามารถปรับเเต่ง UI ได้ เปลี่ยนสีหรือข้อความได้ทั้งหมด!
    - เเละอื่นๆ
</Card>
